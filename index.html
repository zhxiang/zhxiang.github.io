建立jekyll

前提是你已经有了GitHub账号和配置好了git。 建立一个新的repo

名字为USERNAME.github.io, USERNAME 为你的github用户名 克隆本站主题

请将下面操作中的USERNAME替换为你的GitHub用户名

$ git clone git@github.com:pala/pala.github.com.git USERNAME.github.io $ cd USERNAME.github.io $ ./cleanup #will delete all my posts and pages $ git remote set-url origin git@github.com:USERNAME/USERNAME.github.io.git

接着请按需要编辑以下文件：

_config.yml

CNAME (如果不绑定域名请删除)

_layouts/default.html (修改顶部导航超链接)

最后push到github

$ git push origin master
